# Python for Excel with xlwings

This is the material for the course on [YouTube](https://www.youtube.com/playlist?list=PL8T301Ai03xFcE5AFTnfSqnemXAh8cmMR).

This course is from 2018 and is therefore outdated. Nevertheless, you might find it useful as an introduction into the basics.

xlwings is a popular open-source library to program Excel with Python instead of VBA. This course is a series of video tutorials that guide you step by step through the possibilities, lesser known features and pitfalls with xlwings.

The tutorials are split into short videos (usually between 5 and 10 minutes) so that it is easy to quickly find the topics of interest. The course material comes in the form of Jupyter notebooks and is available for download on https://github.com/xlwings/python-for-excel-course.
